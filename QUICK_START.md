### Update Content

1. **Hero Section**: Edit `src/components/Hero.jsx`
2. **About Info**: Edit `src/components/About.jsx`
3. **Portfolio Items**: Edit `src/components/Works.jsx`

### Change Colors

Edit `tailwind.config.js`:
```javascript

  'primary-bg': '#FAF1E8',  // Main background
  'nav-bg': '#FFDADB',       // Navigation bar
}
```

### Add Your Images

Place images in the `Assets/` folder and update component image paths.
